# Monitoring

## Endpoints

- `/api/health`
- `/api/metrics`
- `/api/status`
- `/api/version`

## Health Includes

- uptime
- memory usage when available
- python version
- application version
- frontend version
- environment
- runtime mode
- build info
- startup validation

## Metrics Includes

- build metadata
- uptime
- memory usage
- startup validation snapshot
- lightweight performance diagnostics

Prometheus integration is intentionally out of scope for this EPIC.
