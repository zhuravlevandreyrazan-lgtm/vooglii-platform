# Financial Modes

## 1. Управленческий P&L

Для управления бизнесом.
Используется в `/report`, `/finance`, `/pnl`, `/business`, `/home`.

Этот режим показывает операционную картину бизнеса:
- продажи;
- реклама;
- себестоимость;
- операционные расходы;
- предварительная или подтверждённая прибыль.

## 2. Сверка с официальным отчётом WB

Для проверки совпадения с недельным отчётом WB.
Используется в `/validate` и `/finance validate`.

Этот режим не заменяет управленческий P&L.
Он нужен как контроль качества данных и weekly parity между локальными WB-слоями и официальным weekly report.

## 3. Почему цифры могут отличаться

- используются разные даты признания продаж;
- WB weekly работает по начислениям за неделю;
- Management P&L работает по продажам и операционным расходам;
- реклама, себестоимость и операционная прибыль нужны для управления, но не всегда присутствуют в WB weekly.
